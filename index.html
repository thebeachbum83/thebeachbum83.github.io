About Me

Portfolio

Tribute Project

A tribute page built as part of the FCC front end development program. This was my first foray into web design, and my first time using Bootstrap to create a responsive webpage. I had previously been introduced to JS and JQuery, so I leveraged that knowledge info to change the text on the page.

Random Quote Machine

A random quote machine written also as a part of the FCC front end development program. Here, I began to use animations and different APIs to both retreive the quote, and to post it to Facebook. This was my first attempt at using a JSON driven Restful API.

Local Weather App

A weather app created as another FCC project. This was another exercise in using APIs. Display will vary depending on time of day (day vs night) as well as the weather code coming in from the API. A button was also added to change the temperature units from imperial to metric units.

Wikipedia Search

A wikipedia search engine using the Wikipedia API to gather articles. This was another projecct for FCC. This will display a type ahead lookup of wiki articles and features a number of animations. With this project, I started to branch out more regarding the actual visual design of the application, but kept it simple, yet appealing.

Twitch Viewer

This Twitch viewer was a project for FCC. This takes an array of existing channels and uses RESTFUL API data to gather information regarding those channels and display it to users. This application takes dynamic data and adds style and animations to the different channels available appear based on the user's stream status.
